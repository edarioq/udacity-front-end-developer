==================
Keyboard Shorcuts
==================
Select Word: ⌘ + D
Select Line: ⌘ + L
Select All: ⌘ + A
Duplicate code ⌘ + ⇧ + D with or without text highlighted
Jump to Opening/Closing Parentheses, Brackets, Braces: Ctrl + M
Jump to Definition: ⌘ + R
Reopen Tab: ⌘ + ⇧ + T
Quick Open ⌘ + P
